# Fitness Buddy

## Overview

Fitness Buddy is a full-stack fitness tracking web application that helps users monitor workouts, meals, calorie intake, protein intake, water consumption, and overall fitness progress.

## Features

* User Registration & Login
* Email OTP Verification
* Meal Tracking
* Workout Tracking
* Water Intake Monitoring
* Calories & Protein Calculation
* Dashboard Analytics
* Responsive User Interface

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* PHP

### Database

* MySQL

### Additional Tools

* PHPMailer
* XAMPP

## Database Tables

* users
* meals
* workouts
* food_master
* workout_master

## Project Workflow

1. User registers an account.
2. OTP is sent to the registered email.
3. User verifies OTP and logs in.
4. User records meals and workouts.
5. System calculates calories, protein, and calories burned.
6. Dashboard displays fitness statistics.

## Challenges Faced

* Implementing email OTP verification using PHPMailer.
* Managing database relationships.
* Synchronizing workout and meal data across modules.
* Deploying the application on a hosting platform.
